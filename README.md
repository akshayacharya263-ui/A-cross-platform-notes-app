# Notes App (Flet)

A simple cross-platform notes app built with Python + Flet.

## Features
- Add notes
- Edit notes
- Delete notes
- Search notes
- Local save using `notes.json`

## Run
1. Install Python 3
2. Open terminal in this project folder
3. Install dependency:
   pip install -r requirements.txt
4. Run:
   python main.py
